# ssg poc time

## Installatie

Volg deze stappen om je project lokaal te installeren en draaien:

1. Clone de repository:
    ```bash
    git clone https://github.com/Jahmiro/ssg-poc-time
    ```
2. Navigeer naar de projectmap:
    ```bash
    cd ssg-poc-time
    ```
3. Installeer de benodigde dependencies:
    ```bash
    npm install
    ```
    of
    ```bash
    yarn install
    ```

## ssg Testen

Om ssg te testen:
